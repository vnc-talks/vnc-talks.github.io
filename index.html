About this blog

This site is fully static and consumes markdown files from the posts/ folder. Add a new entry to posts/index.json, drop in a markdown file, and it will appear automatically.

Typography is set in Inter with relaxed line heights for long-form reading. The layout is responsive and keeps a single-column flow on small screens, while spacing grows subtly on large displays.

  • Supports headings, paragraphs, lists, links, code blocks, and quotes.
  • No build steps required; open index.html locally or serve with any static host.
  • Accessible focus states, prefers reduced motion when requested by the user.
